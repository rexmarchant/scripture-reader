How It Works
1Drop your V2 SSML file — the one with verse numbers retained, used for the sync player.
2Drop the Whisper VTT file generated from the same chapter's MP3.
3Click Convert — each verse's start time is located in the Whisper transcript by matching the first words of that verse.
4Download the output verse-aligned VTT and drop it into the Audio Sync Player instead of the SSML. Perfect timestamps, no manual playback needed.
